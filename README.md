# Short Description

Converting the JPG/PSD layout into HTML, CSS using SASS and responsive layout without bootstrap. Project is purely done in HTML5.

## How to run the project?

- Clone the repository using HTTPS.
- Run `npm install` (This will install the necessary dependencies for the project).
- Double click on `index.html` file within the public folder of the directory to access the project in your default browser `OR` check into the current project directory in `cmd` and run `node server.js`, keep it running and then open your choice of browser. And type `localhost:3000` in URL and press enter.
